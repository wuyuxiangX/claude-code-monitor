# Changelog

## 1.0.0 — Initial Release

### Features

- Real-time session monitoring with status tracking (Active, Waiting, Idle, Ended)
- Menu bar icon with live session count and color-coded status
- Usage dashboard with daily, weekly, and monthly cost/token breakdowns
- Extensions manager for plugins, skills, and MCP servers
- One-click hook setup for automatic session detection
- AI-generated session labels for quick identification
- Multi-terminal support: VS Code, Cursor, Zed, iTerm2, Warp, Ghostty, and more
- Session focus and resume actions
- Git branch display and worktree detection
- Per-project and per-model usage breakdown
